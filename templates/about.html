About

The Court Case Info Dashboard is a web-based tool designed to help users quickly retrieve case metadata from Indian court portals. This includes parties involved, case status, important hearing dates, and more — all by simply entering the case type, number, and year.

Whether you're a law student, researcher, or a curious citizen, this tool provides a fast and user-friendly interface for legal data tracking. It's built using Python (Flask), web scraping with Selenium, and an SQLite backend for query logging.
